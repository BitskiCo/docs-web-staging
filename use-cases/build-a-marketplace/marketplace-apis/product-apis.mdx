---
title: "Overview"
description: "Use the Bitski Product APIs to get public information about published products
that are for sale on Bitski's marketplace."
---

You can query based on:

- a specific creator
- a specific product
- return all live products

---

<Note>
**No auth needed**

You don't need an authentication token to use these endpoints.

</Note>

---

<Info>
See it in action

These Product APIs power Bitski's own
[marketplace](https://www.bitski.com/marketplace). Check the marketplace to see
them in action.

</Info>
