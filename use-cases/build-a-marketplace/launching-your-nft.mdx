#### What specs should I keep in mind when designing my NFT?

**_Primary NFT Image (include even if the NFT is a video)_**

- 10MB Max size
- PNG, JPG or GIF

**_Primary NFT Video_**

- 50MB Max Size
- .MP4
- 1080 x 1080

**_Unlockable Content (Optional)_**

**High Res Image**

- PNG or JPG
- 50MB Max Size
- Any dimensions

**_4K Video_**

- .MP4
- 200MB Max Size
- Any aspect ratio

**_Snap Lens_**

- Upload image of snapcode

**_Instagram Filter_**

- Use instagram Filter URL

**_Any Additional Files_**

- additional videos
- 3D models (.GLB)
- Wallpapers
- .ZIP Files
- 500MB Max Size
