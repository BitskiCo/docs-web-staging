---
title: "Get token metadata"
description:
  "Token metadata returns the associated metadata for a non-fungible (ERC721) or
  semi-fungible (ERC1155) token on the Ethereum or Polygon blockchains."
api:
  "GET
  https://api.bitski.com/v1/tokens/{coinType}/{chainId}/{contractAddress}/{tokenId}"
---

### Path Params

<ParamField path="contractAddress" type="string" required>
  The contract address to get data done.
</ParamField>

<ParamField path="tokenId" type="string" required>
  The token id, in decimal or hex format, to fetch.
</ParamField>

<ParamField path="chainId" type="string" required>
  Set the network to get data from. For EVM chains, see a list of available
  chains here.
</ParamField>

<ParamField path="coinType" type="string" required>
  Set the blockchain to get data from. Right now, only EVM based blockchains are
  supported.

</ParamField>

### Query Params

<ParamField query="forceRefresh" type="string" required>
  The contract address to get data from.
</ParamField>
