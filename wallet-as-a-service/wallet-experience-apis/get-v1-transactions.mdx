---
title: "GET Transactions"
api: "GET https://api.bitski.com/v1/transactions/{transactionId}"
description: "Optionally prompt users for approval when signing. Transaction approval screen simulates transaction effects, parses complex transaction results, and displays content in human-readable terms."
---

<Note>Access to this API requires authentication. To receive help on accessing this API, please reach out to support@bitski.com.</Note>

### Path
<ParamField path="transactionId" type="string" placeholder="5c94e001-a9d3-4253-ac43-663739e29c43" required>
  The ID of the transaction to retrieve.
</ParamField>
