---
title: "GET Products"
api: "GET https://api.bitski.com/v1/products"
description: "Get a list of all the live products for a storefront."
---

### Parameters

<ParamField query="baseUrl" type="string" placeholder="https://www.bitski.com/@bitski" optional>
  URL for a Bitski creator, e.g. https://www.bitski.com/@username, to filter
  products by. By default, uses the current user's first storefront in the list.
</ParamField>

<ParamField query="saleType" type="string" optional>
  Which type of product listing to return. 
  One of `EDITION`, `LIMITED_EDITION`, `OPEN_EDITION`, `AUCTION`, `OFF_CHAIN_AUCTION`, `ON_CHAIN_AUCTION`
</ParamField>

<ParamField query="sold" type="boolean" optional>
  Filter by if the product is sold (true) or not (false).
</ParamField>

<ParamField query="all" type="boolean" optional>
  Filter by all products.
</ParamField>

<ParamField query="includeUpcoming" type="boolean" optional>
  Include upcoming products scheduled to go live.
</ParamField>

<ParamField query="liveAt" type="string" optional>
  The date a product or list of product goes live.
</ParamField>

<ParamField query="limit" type="integer" optional default="0">
  Limit the number of products returned.
</ParamField>

<ParamField query="offset" type="integer" optional default="0">
  Product to start from when paginating.
</ParamField>
